# Roadmap do Projeto 🗺️

Este roadmap descreve as fases de aprendizado e desenvolvimento das competências no uso de ferramentas essenciais para o mercado de ADS.

## Fase 1: Fundamentos e Organização (Aulas 01-04)
- [ ] **Introdução**: Entender o ecossistema de ferramentas modernas.
- [ ] **Gestão**: Configurar quadros Kanban e Scrum (Jira/Trello).
- [ ] **Ambiente**: Customização de IDEs (VS Code) e domínio do Terminal.
- [ ] **Git**: Fluxos de trabalho básicos e versionamento local.

## Fase 2: Colaboração e Fluxo de Dados (Aulas 05-08)
- [ ] **Plataformas**: Pull Requests, Code Review e colaboração no GitHub/GitLab.
- [ ] **Dados**: Manipulação de bases SQL e NoSQL via clientes visuais.
- [ ] **Qualidade Inicial**: Escrita de testes básicos para garantir estabilidade.

## Fase 3: Automação e Qualidade (Aulas 09-11)
- [ ] **APIs**: Testes automatizados e manuais de endpoints com Postman.
- [ ] **Linting**: Padronização de estilo e correção estática de código.
- [ ] **CI/CD**: Criação de workflows automáticos para build e teste.

## Fase 4: Infraestrutura e Handoff (Aulas 12-16)
- [ ] **IaC & Automação**: Configuração de servidores via código (Ansible).
- [ ] **Contêineres**: Dockerização completa de aplicações.
- [ ] **Orquestração**: Noções básicas de Kubernetes para escala.
- [ ] **Design & Soft Skills**: Comunicação em equipe e uso do Figma.

## 🚀 Próximos Passos (Manutenção)
- [x] Deploy GitHub Pages (GitHub Actions)
- [ ] Atualização para novas versões das ferramentas (Docker/Git/VS Code)
- [ ] Inclusão de exemplos de ferramentas de monitoramento (Prometheus/Grafana)
