# Setup 03: Ferramentas de Apoio 🛠️

Além do ambiente de desenvolvimento principal, algumas ferramentas são indispensáveis para o fluxo de trabalho profissional.

## 1. Controle de Versão (Git)
O Git é essencial para versionar e colaborar no código.
1.  **Instalação**: Baixe em [git-scm.com](https://git-scm.com/).
2.  **Configuração Inicial**:
    ```bash
    git config --global user.name "Seu Nome"
    git config --global user.email "seu@email.com"
    ```

## 2. Terminal Moderno
Um bom terminal aumenta sua produtividade.
*   **Windows**: Instale o **Windows Terminal** via Microsoft Store.
*   **Dica**: Utilize o **Oh My Posh** ou **Starship** para ter um prompt visual e informativo.

## 3. Testes de API (Postman/Insomnia)
Ferramentas para interagir com servidores e testar endpoints.
1.  **Postman**: Baixe em [postman.com/downloads](https://www.postman.com/downloads/).
2.  **Insomnia**: Baixe em [insomnia.rest/download](https://insomnia.rest/download).
3.  **Uso**: Crie uma conta para sincronizar suas coleções de testes na nuvem.

## 4. Navegador para Desenvolvedores
*   **Google Chrome** ou **Firefox Developer Edition**: Utilize as "DevTools" (F12) para inspecionar redes e elementos de interface.
