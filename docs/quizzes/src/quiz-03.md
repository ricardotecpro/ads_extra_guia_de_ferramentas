# Quiz: Aula 03 - Ambiente de Desenvolvimento 💻

1.  **Qual a principal vantagem de usar o VS Code como seu editor principal?**
    - [ ] Ele substitui a necessidade de programar.
    - [x] Ele é leve, gratuito e possui um ecossistema gigantesco de extensões.
    - [ ] Ele só funciona em computadores muito caros.
    - [ ] Ele apaga o código sozinho se houver erros.
    *   *Explicação: O VS Code se tornou padrão de mercado por ser altamente customizável e eficiente.*

2.  **O que o comando `ls` faz no terminal (Linux/Mac) ou `dir` (Windows)?**
    - [ ] Deleta um arquivo.
    - [x] Lista os arquivos e pastas do diretório atual.
    - [ ] Cria um novo arquivo.
    - [ ] Formata o computador.
    *   *Explicação: Esses comandos são usados para "olhar" o que existe dentro da pasta onde você está.*

3.  **Para que serve a extensão "Prettier" no VS Code?**
    - [ ] Para traduzir o código para português.
    - [x] Para formatar o código automaticamente, deixando-o padronizado e bonito.
    - [ ] Para enviar o código para o GitHub.
    - [ ] Para mudar o papel de parede do computador.
    *   *Explicação: O Prettier foca no visual e padronização do código (espaços, aspas, etc.).*

4.  **Qual o comando de terminal usado para entrar em uma pasta?**
    - [ ] `move <pasta>`
    - [x] `cd <pasta>`
    - [ ] `enter <pasta>`
    - [ ] `goto <pasta>`
    *   *Explicação: `cd` significa "Change Directory" (Mudar de Diretório).*

5.  **O que acontece se você digitar `mkdir projeto` no terminal?**
    - [ ] Ele deleta a pasta "projeto".
    - [x] Ele cria uma nova pasta chamada "projeto".
    - [ ] Ele abre o navegador na pasta "projeto".
    - [ ] Ele renomeia o arquivo atual para "projeto".
    *   *Explicação: `mkdir` significa "Make Directory" (Criar Diretório).*

6.  **O que é o Integrated Terminal (Terminal Integrado) no VS Code?**
    - [ ] Um terminal que só funciona offline.
    - [x] Uma janela de terminal que abre dentro do próprio editor, evitando trocar de janelas.
    - [ ] Um terminal que corrige erros de português.
    - [ ] Uma extensão paga do VS Code.
    *   *Explicação: Ter o terminal dentro do editor aumenta muito a produtividade do desenvolvedor.*

7.  **Qual a função dos "Keyboard Shortcuts" (Atalhos de Teclado)?**
    - [ ] Deixar o teclado mais colorido.
    - [x] Executar tarefas comuns (salvar, buscar, abrir) sem precisar usar o mouse.
    - [ ] Substituir o monitor.
    - [ ] Bloquear o acesso de estranhos ao código.
    *   *Explicação: Dominar atalhos separa o desenvolvedor iniciante do profissional ágil.*

8.  **Sobre o uso do terminal, é correto afirmar:**
    - [ ] É algo perigoso que só hackers devem usar.
    - [x] É uma habilidade essencial, pois muitas ferramentas de dev só funcionam via linha de comando.
    - [ ] É mais lento do que usar o mouse para todas as tarefas.
    - [ ] Não funciona no Windows.
    *   *Explicação: O terminal é a "casa" do desenvolvedor e o local onde as automações acontecem.*

9.  **O que significa "IntelliSense" no VS Code?**
    - [ ] Um tipo de vírus.
    - [x] O recurso de completagem inteligente de código e sugestões de sintaxe.
    - [ ] Uma versão do VS Code para tablets.
    - [ ] Um sistema de inteligência artificial que escreve todo o projeto sozinho.
    *   *Explicação: O IntelliSense ajuda a escrever código mais rápido e com menos erros de digitação.*

10. **Como você abre a "Command Palette" (Paleta de Comandos) no VS Code?**
    - [ ] `Alt + F4`
    - [ ] `Esc`
    - [x] `Ctrl + Shift + P` (ou `Cmd + Shift + P`)
    - [ ] `Space + Enter`
    *   *Explicação: A paleta de comandos dá acesso a quase todas as funções do editor rapidamente.*
