# Projeto 02 - Criando seu Primeiro Quadro Ágil 📊

### 🎯 Objetivo
Organizar um fluxo de trabalho utilizando o método Kanban para um projeto fictício de desenvolvimento.

### 📋 Passo a Passo

1.  **Escolha da Ferramenta**: Acesse o **Trello** ou utilize as **GitHub Issues** do repositório criado no Projeto 01.
2.  **Criação das Colunas**: Configure seu quadro com as colunas: `Backlog`, `To Do`, `Doing`, `Done`.
3.  **Criação de Cards**: Adicione 5 tarefas necessárias para criar um site simples (ex: "Criar HTML", "Estilizar com CSS", "Testar no Chrome").
4.  **Atribuição e Prazo**: Em pelo menos 2 cards, adicione uma data de entrega (Due Date) e uma etiqueta de prioridade (ex: "Alta", "Baixa").
5.  **Movimentação**: Mova 2 tarefas para `Doing` para simular o início do trabalho.

---
**Entrega**: O link público do seu quadro no Trello ou o link das Issues no seu GitHub.
