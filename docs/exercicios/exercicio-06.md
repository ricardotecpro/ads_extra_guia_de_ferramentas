# Exercícios: Aula 06 - DB Relacional 💾

### 🟢 Básicos (Fixação)

1.  **Criação de Tabelas**: Escreva o comando SQL (ou descreva os campos na interface visual) para criar uma tabela chamada `Alunos` com os campos: `Matricula` (Número), `Nome` (Texto) e `Curso` (Texto).
2.  **Consulta Simples**: Como você faria para listar apenas os nomes de todos os alunos que estão no curso de "ADS" usando SQL?

### 🟡 Intermediários (Aplicação)

3.  **Relacionamentos**: Imagine um sistema de biblioteca. Você tem a tabela `Livros` e a tabela `Autores`. Como você faria para "ligar" um livro ao seu autor? Explique o conceito de **Foreign Key** (Chave Estrangeira).
4.  **DBeaver Workflow**: Descreva os passos para exportar os dados de uma tabela para um arquivo CSV usando a interface do DBeaver. Por que essa funcionalidade é útil para um desenvolvedor?

### 🔴 Desafio (Exploração)

5.  **Performance de Consulta**: Se uma tabela de `Vendas` tiver 10 milhões de linhas, a busca por uma data específica pode ficar lenta. Pesquise o que é um **Índice (Index)** em bancos de dados SQL e como ele ajuda a acelerar essa busca.
